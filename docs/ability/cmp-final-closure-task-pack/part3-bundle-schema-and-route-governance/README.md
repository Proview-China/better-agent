# Part 3 Bundle Schema And Route Governance

## 目标

把 `core return / child seed / peer exchange` 三类包彻底 schema 化，并把路由治理规则做实。

## 子任务

1. 三类包共同主干字段
- `target`
- `body`
- `governance`
- `source anchors`

2. `core return`
- 层次：
  - 正文
  - 引导
  - skill
  - timeline

3. `child seed`
- 层次：
  - 子任务
  - 边界
  - 引导
  - 背景

4. `peer exchange`
- 层次：
  - 协作意图
  - 依赖变更
  - 局部正文
  - 引导

5. 路由治理
- peer approval 双挂
- child seed 强拦截到 child ICMA
- 允许极少数人工放行
